# FullCare
FullCare on Web-palvelu, jossa asiakas voi varata vastaanotto ajan lääkärille. Vastaanottoaikoja on tarjolla useilla eri paikkakunnilla. Palvelun käyttäminen edellyttää rekisteröitymistä.
